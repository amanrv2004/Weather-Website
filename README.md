A simple and beautiful weather application built using HTML, CSS, and JavaScript. Enter your city name to get the current temperature, humidity, and wind speed using the OpenWeatherMap API.


🛠 Features : 
1.  Search weather by city name
2.  Displays current temperature in Celsius
3.  hows humidity and wind speed
4.  Dynamic weather icons (Clear, Clouds, Rain, Snow, etc.)
5.  Error handling for invalid city names
6.  Responsive design for mobile and desktop
7.  Search on button click or Enter key
